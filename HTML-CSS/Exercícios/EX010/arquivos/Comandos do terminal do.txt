Comandos do terminal do Linux

sudo: Comando de administrador

apt: Comando para gerenciar pacotes (instalar, remover, etc)

install: Comando para instalar pacotes

remove: Comando para remover pacotes ex: sudo apt remove pacotetal

cd: Comando para manipular diretórios ex: cd Downloads/

update: Comando para atualizar pacotes 

touch: Comando que cria arquivos

ls: Lista todos os arquivos do diretório

df: Mostra a quantidade de espaço usada no disco rígido
   
top: Mostra o uso da memória

cd: Acessa uma determinada pasta (diretório)
    
mkdir: Cria um diretório
   
rm: Remove um arquivo/diretório

rmdir: remove o diretorio
    
cat: Abre um arquivo
    
vi: Abre o editor vi (lê-se viai) para editar/criar arquivos

moo: Comando especial

nano: Cria om bloco de notas

cp: Copia arquivos

mv: Move o arquivo

apropos: Localiza comandos por pesquisa de palavra-chave
    f
ind: Localizar arquivos, como por exemplo: find . -name *.txt -print, para pesquisa de arquivos de texto do diretório atual
    
    
info: Abre o explorador de informações
    
man: Manual muito completo, pesquisa informação acerca de todos os comandos que necessitemos de saber, como por exemplo man find
    
whatis: Descreve o que um determinado comando é/faz
    
whereis: Localizar a página de ajuda (man page), código fonte, ou arquivos binários, de um determinado programa
